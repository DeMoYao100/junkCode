# junkCode
###### 简单的花指令批量生成
#### 使用方法：
##### 直接修改源码路径，生成加花源码。编译器用的是`MinGW GCC 11.2.0 64-bit Debug`(Release 版本下可能出现奇怪的报错，更换编译器对应版本自行需要修改asm写法，具体修改参考网上内联汇编一般的加花格式即可)
##### 源码的函数写法
``` cpp
returnValue func(){ //这里`{`和`)`相连即可，也可以自行修改
}
```
#### 使用效果：
##### 对应源码的每个函数开头随机一段距离生成随机一个花指令（原理相同，具体结构随机）
